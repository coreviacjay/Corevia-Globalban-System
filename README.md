# Corevia Global Ban System Bot
A fully automated, network‑wide Global Ban System for Discord.  
Designed for multi‑server moderation teams that need consistent enforcement across all servers the bot is in.  
Built with Corevia‑grade structure, modular commands, and role‑locked permissions.


## 🚀 Features

### 🌐 Global Ban System (Auto‑Sync)
- `/globalban-add`  
  - Adds a user to the global ban list  
  - Automatically bans them across **every server** the bot is in  

- `/globalban-remove`  
  - Removes a user from the global ban list  
  - Automatically unbans them across **every server** the bot is in  

### 🔄 Sync Ban System
- `/syncban`  
  - Manually sync‑ban a user across all servers  

### 🔐 Staff‑Only Access
All commands are locked to a **specific staff role**, configured in `config.json`.

### 📁 Organized Command Structure
Each command is in its own file for clean GitHub organization.
